## Map generation

# Enables hills for the flat mapgen.
# Currently experimental and will break your world.
pcity_enable_hills (Enables hills for the flat mapgen) bool false

# Citychunk size in the X dimension stated in mapchunks. This setting affects the size of the basic city generation unit.
# The higher the value the more dispersed the road system.
# Sizes smaller than 3 are not recommended and will slow down map generation.
pcity_citychunk_size_x (Citychunk size X) int 10 1

# Citychunk size in the Y dimension (height) stated in mapchunks.
# This allows for different vertical scaling of city chunks compared to horizontal.
# A smaller value means shorter vertical chunks (faster generation but less vertical variation).
pcity_citychunk_size_y (Citychunk size Y) int 2 1

# Citychunk size in the Z dimension stated in mapchunks.
# This setting affects the size of the basic city generation unit along the Z axis.
# Sizes smaller than 3 are not recommended and will slow down map generation.
pcity_citychunk_size_z (Citychunk size Z) int 10 1

# Enable automated testing for Perfect City mods.
# When enabled, mods will run their unit tests on startup and shut down after completion.
# This should be disabled for normal gameplay and only enabled for development/testing.
pcity_run_tests (Run automated tests) bool false
